# Projeto-TCC-Senai

## Objetivos do Repositório
O tema deste projeto foi escolhido com base nas demandas cadastradas por empresas na plataforma SAGA SENAI, uma plataforma que tem por objetivo permitir que os alunos resolvam demandas reais do mercado de trabalho.

[cite_start]Escolhemos a demanda **Mapeamento de Competências e Aptidões Internas** (ID da Demanda: 12276) [cite: 8][cite_start], que visa atender a empresa **PAREX Indústria Mecânica, Caldeiraria e Estruturas Metálicas S.A.** [cite: 9] no seguinte desafio:

> [cite_start]"A empresa Parex necessita identificar talentos e competências de forma estruturada para otimizar a distribuição de funções e o desenvolvimento profissional. [cite: 11] [cite_start]Nossa solução consiste no desenvolvimento de um sistema de mapeamento via "Quiz Vocacional". [cite: 12] [cite_start]O colaborador responderá a um formulário interativo de competências técnicas e comportamentais (soft skills), e o sistema processará os dados para gerar um banco de talentos, sugerindo a melhor alocação ou trilha de treinamento para cada perfil." [cite: 13]

## Sobre o Desafio
* [cite_start]O mapeamento de competências e aptidões internas permite identificar de forma estruturada os conhecimentos, habilidades, comportamentos e potenciais dos colaboradores. [cite: 26]
* [cite_start]Isso possibilita compreender talentos, otimizar a distribuição de funções, apoiar decisões de desenvolvimento profissional e fortalecer a gestão de pessoas, gerando mais eficiência, alinhamento estratégico e melhor aproveitamento das capacidades individuais e coletivas. [cite: 27]

## Tecnologias e Metodologia
[cite_start]A construção da ferramenta digital proposta baseia-se em uma arquitetura de software dividida nas camadas de Frontend, Backend e Banco de Dados, estruturada para atender a dois perfis de usuários distintos: o Colaborador e o Gestor de RH. [cite: 121]

* [cite_start]**Front-End:** O sistema utilizará HTML5 e CSS3 para criar uma interface de quiz moderna e amigável, reduzindo a resistência dos colaboradores. [cite: 16] [cite_start]JavaScript será utilizado para gerenciar a lógica em tempo real e renderizar gráficos dinâmicos para o colaborador. [cite: 17, 124]
* [cite_start]**Back-End:** O sistema será desenvolvido utilizando a linguagem PHP para processar os algoritmos de mapeamento e gerar diagnósticos individuais. [cite: 126, 127]
* [cite_start]**Banco de Dados:** Utilizaremos SQLite para armazenar as informações localmente de forma estruturada. [cite: 21, 131]
